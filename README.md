### Как развернуть локально

1. Склонировать репозиторий

```
git clone https://github.com/Vova-Pol/portfolio.git
```

2. Установить зависимости

```
npm i
```

3. Запустить проект

```
npm run start
```

4. Готово! Проект работает локально по адресу http://localhost:3000/portfolio
